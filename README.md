# project
По задумке, наш прокет составляет телеграмм бота и сайт, для подготовки к Олимпиаде по информатике. Тг бот - размещены пробные варианты задачек с решениями Олимпиады, гайды, полезные и интересные штучки для изучения информатики. Сайт - дополнение к боту, в нем будут собраны самые точные, понятные и Мега интересные уроки для изучения языков и решения всяких задачек, а так же доступ к др полезным курсам. На данный момент проект представляет из себя прекрасную идею помощи для подготовки к Олимпиаде, благодаря этому проекту к Олимпиаде станет готовиться не только легко и удобно, а еще увлекательно и интересно, человеку использующему данную в уже реализованную задумку будет очень нравится весь процесс! Таким образом количество умных людей, заинтересованных в IT сфере станет в разы больше, что приведет к большому успеху!
